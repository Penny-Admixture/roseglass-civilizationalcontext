## Slide 1: HOME COMPUTER 99/8 — INTRODUCTION

- THE 99/8 IS ON SCHEDULE FOR FIRST SHIPMENTS IN SEPTEMBER

- HOWEVER, THE 99/8 WILL NOT BE INTRODUCED AND SHOWN ON THE FLOOR. WE WILL LOSE MORE THAN WE GAIN BY INTRODUCING PRODUCTS SEVERAL MONTHS BEFORE SHIPMENT, THEREFORE WE ARE BEING MORE CONSERVATIVE ABOUT NEW PRODUCT INTRODUCTIONS

- WORKING 99/8S ARE SET UP IN THE BACK ROOM FOR DEMONSTRATION TO MAJOR CUSTOMERS

- GRAND INTRODUCTION IS BEING PLANNED FOR SEPTEMBER WITH A PRESS CONFERENCE IN NEW YORK WHICH WILL BE TRANSMITTED BY SATELLITE TO MAJOR CITIES

## Slide 2: HOME COMPUTER 99/8 — HARDWARE FEATURES

**KEYBOARD:**
- 49 KEYS
- SOFTWARE LOCK FOR CAPITALS ONLY ENTRY
- FUNCTION KEY USED FOR PROGRAM DEFINED KEYS ONLY

**MEMORY:**
- 80K BYTES OF RAM: 64K FOR PROGRAMS, 16K FOR VIDEO
- 228K BYTES OF ROM
- MEMORY EXPANSION UP TO 16 MEGA BYTES

**INTERFACES:**
- HEXBUS INTERFACE FOR USE OF ALL TI HEXBUS PERIPHERALS
- 50 PIN HIGH SPEED INTERFACE FOR MEMORY EXPANSION AND WINCHESTER DISK DRIVES
- AUDIO CASSETTE INTERFACE IDENTICAL TO 99/4AS
- JOYSTICK PORT FOR USE OF TIS JOYSTICKS AND MILTON BRADLEY EXPANDER
- EXTERNAL MODULATOR FOR USE WITH T.V. SETS AND DIRECT COMPOSITE VIDEO OUT FOR MONITORS

# TI Home Computer 99/8 — Internal Documents

**Classification at time of creation**: TI STRICTLY PRIVATE  
**Date**: 06-04-83  
**Reference**: HMS 039-041  
**Subject**: Texas Instruments Home Computer 99/8 pre-release briefing  

## Historical Context
The TI-99/8 was the planned successor to the TI-99/4A home computer.  
It was never commercially released — TI exited the home computer market in late 1983,  
just months after these documents were created.  
These slides are from an internal sales/dealer briefing circa June 1983.

## Why It Is Here
Primary source historical computing ephemera. Captures the exact moment  
a product line was about to launch and then was killed. The gap between  
"on schedule for first shipments in September" and TI exiting the market  
that same year is roughly four months.

## Key Facts from Documents

### Introduction Slide
- 99/8 on schedule for first shipments in September [1983]
- Would NOT be shown on the trade show floor — TI being "more conservative about new product introductions"
- Working 99/8s set up in back room for demonstration to major customers only
- Grand introduction planned for September with press conference in New York, transmitted by satellite to major cities

### Hardware Features Slide
**Keyboard**: 49 keys, software lock for capitals-only entry, function key for program-defined keys only

**Memory**:
- 80K bytes RAM: 64K for programs, 16K for video
- 228K bytes ROM
- Memory expansion up to 16 megabytes

**Interfaces**:
- Hexbus interface for all TI Hexbus peripherals
- 50-pin high speed interface for memory expansion and Winchester disk drives
- Audio cassette interface identical to 99/4As
- Joystick port for TI joysticks and Milton Bradley Expander
- External modulator for TV sets + direct composite video out for monitors

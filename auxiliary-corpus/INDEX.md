## USAGE

**To add a new document:**
1. Upload source to `/auxiliary-corpus/raw/`
2. Analyze with available models
3. Save analyses to `/auxiliary-corpus/analysis/`
4. Update this INDEX.md with entry information
5. Cross-reference with main archive where applicable

**For images/PDFs:**
1. Upload original to `/raw/original_filename.pdf`
2. Extract text via OCR
3. Save extracted text as `/raw/original_filename_extracted.md`
4. Track both versions in INDEX
